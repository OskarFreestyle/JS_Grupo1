Características Avanzadas: Temporizadores

Un Temporizador es un reloj o contador que podemos utilizar para realizar una serie de acciones tras un tiempo determinado.

En la pestaña Temporizadores podemos agregar nuevos Temporizadores pulsando en . Una vez tengamos uno, si lo seleccionamos podremos editar el Tiempo que queremos que dure, si queremos que se muestra la cuenta durante el juego, seleccionando Mostrar en juego y otras opciones:

DOCUMENTACIÓN DEL TEMPORIZADOR

Información acerca del temporizador seleccionado. Es sólo informativo para las tareas de desarrollo. No se utilizará en el motor.

TIEMPO

Este apartado sólo aparece activo si hemos seleccionado Mostrar en juego. Las opciones son:

  • Nombre a mostrar: El nombre que queremos que aparezca junto al reloj.
  • Cuenta regresiva: Si está activa, el paso del tiempo será regresivo (Desde el tiempo marcado hasta 0) cuando se muestre en pantalla.
  • Mostrar cuándo está parado: Si está activa, el contador se mostrará siempre, aunque no esté activo.

CONTROL DE CICLOS

En este apartado controlamos el comportamiento del temporizador cuándo se detiene o cumple un ciclo.

  • Multiples inicios:
    • Activo: Si el temporizador está parado, cuándo se cumplan las condiciones de inicio comenzará nuevamente desde cero.
    • Inactivo: Si el temporizador está parado no vuelve a empezar incluso aunque se cumplen las condiciones de inicio.
  • Utiliza bucles:
    • Activo: Si transcurre el tiempo del temporizador, este entrará en un bucle una y otra vez hasta ser detenido.
    • Inactivo: Cuándo transcurre el tiempo, el temporizador se detiene.

CONDICIONES PARA ACTIVACIÓN DEL TEMPORIZADOR

En este apartado editamos las condiciones que inician el temporizador. Más información sobre condiciones.

CONDICIONES PARA LA DESACTIVACIÓN DEL TEMPORIZADOR

Si marcamos Utiliza condiciones para desactivación el temporizador se detendrá cuándo se cumplan las condiciones aquí definidas. En el caso de no haber marcado esa casilla, el temporizador se detendrá cuándo dejen de cumplirse las condiciones iniciales.

  • Efectos a ejecutar si se desactiva el temporizador
