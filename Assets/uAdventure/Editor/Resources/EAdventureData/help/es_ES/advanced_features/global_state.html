Caracterísitcas avanzadas: Estado global

Un estado global es un conjunto de condiciones. Se dice que el juego está en un estado global si se cumplen las condiciones definidas en éste.

Así mismo, podemos utilizar los estados globales directamente como condiciones.

Una vez hemos añadido un estado global () podemos editar su Información y la Lista de condiciones del estado global.
