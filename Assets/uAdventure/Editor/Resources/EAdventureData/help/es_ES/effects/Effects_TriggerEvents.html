Lanzar Eventos

También existen efectos que nos permiten lanzar determinados eventos.

Los efectos que se encuentran en este grupo son los siguientes:

  • Lanzar conversación: mediante este efecto se lanza una conversación.
  • Lanzar escena intermedia: mediante este efecto se lanza un escena de corte, pudiendo mostrar información como un conjunto de diapositivas o mediante un vídeo.
  • Disparar libro: se muestra información en forma de libro.
  • Lanzar escena: mediante este efecto se realiza un cambio de escena.
  • Lanzar última escena: mediante este efecto de pasa a la escena final.
