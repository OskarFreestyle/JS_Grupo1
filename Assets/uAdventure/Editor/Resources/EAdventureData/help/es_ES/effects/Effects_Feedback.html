Efectos de Notificación

Los efectos de notificación permiten dar información al usario de distintas maneras. Este tipo de efectos son utilizados para dar la noción de feedback por parte del juego al usuario.

Los efectos que se encuentran en este grupo son los siguientes:

  • Protagonista dice: el protagonista dice el texto que se introduzca para este efecto.
  • Personaje dice: el personaje dice el texto que se introduzca para este efecto.
  • Lanzar libro: se muestra información en forma de libro.
  • Mostrar texto: se muestra el texto introducido en la posición elegida, y con la combinación de colores seleccionada. Este texto no es reproducido por ningún personaje, puede considerarse como una voz en "off".
  • Lanzar una escena intermedia: mediante una escena de corte se proveé de información al usuario. Estas escenas pueden ser de diapositivas o de vídeo.
  • Lanzar una conversación: proveer información al usuario mediante el desarrollo de una conversación.
