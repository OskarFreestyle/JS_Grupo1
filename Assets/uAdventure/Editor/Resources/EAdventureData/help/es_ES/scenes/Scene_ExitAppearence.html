Apariencia de la salida

Texto de la salida

El texto de la salida es el que se muestra cuando el usuario pone el ratón encima.

Cursor

El cursor que se muestra es el que aparecerá en el ratón cuando el usuario esté sobre la salida. Con el botón "Select", se puede elegir el cursor. Con el botón de la cruz, se puede quitar el cursor seleccionado. Siempre se muestra un cursor, cuando no hay ninguno seleccionado se utilizará el establecido por defecto.
