Metadatos LOM-ES

Conjunto de metadatos siguiendo el estándar LOM-ES. LOM-ES es una adaptación del estándar IEEE-LOM para la comunidad educativa española. Entre los cambios más significativos está la ampliación de los términos de algunos vocabularios, cardinalidades y obligatoriedad de algunos elementos.En este editor se permite modificar los elementos marcados como obligatorios por el estándar. Los juegos exportados como Objetos de Aprendizaje del tipo AGREGA llevarán los metadatos rellenados en este editor. En caso de querer ampliar la edición, se deberá exportar el juego como "AGREGA" y utilizar la herramienta de catalogación. Para más información al respecto, consultar la página www.educaplus.org/documentos/lom-es_v1.pdf.
