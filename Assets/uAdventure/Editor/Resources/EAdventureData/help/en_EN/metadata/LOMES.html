LOM-ES Metadata

Metadata set LOM-ES compliant.

LOM-ES is an adaptation of the IEEE-LOM for the educational Spanish community. Among the most significant changes is the extension of terms of some vocabularies, cardinality and mandatory elements.

This editor lets you modify the elements marked as required by the standard. The games exported as Learning Objects type AGREGA will have the metadata filled in this editor. If you want to make broader the editionissue, the game must be exported as "AGREGA" and then use the documentation tool..

For more information, check out the web site www.educaplus.org/documentos/lom-es_v1.pdf.
