Character: Actions

Actions table

We can define actions and their consequences (effects) for each character. To add a new action you click on . Availables actions ar

  • Normal actions: with Examine, Use and Talk to (usually used to launch a conversation).
  • Custom action: Custom action can be adapted to fit different requirements. Once selected, a dialog will ask us for a name for the action, (e.g.: buy, open, throw, etc. ). A personalize appearence can be given to this actions in the Personalization tab

Each action is defined in the same way, beyond some effects that will be explained later and some conditions that appear in the top table:

  • Needs get to: if it is necessary to reach and item to perform an action. We can set the minimum distance between the item and the main character necessary to perform the action. This condition is only used in third person adventures.
  • Conditions: This defines the conditions which must be checked before the action is performed. If the conditions are not met, nothing will happen. More information about conditions.

Description and effects

Each action has a Full description for the action and Effects.

In the first, we can add all the information we want about the action.

In the second, clicking on we can add Effects for the action. More about effects.

Also, selecting Active when conditions are not and then clicking on Edit not-effects we can define the effects that will be triggered when we perform the action but the conditions defined in top table are not met.
