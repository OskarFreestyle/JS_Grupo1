Character: Dialog configuration

In the Dialog configuration tab we can define the dialog configuration of the Character.

Colors of the player's conversation lines

In this place, we define the dialog appearance:

  • Font front color: color for the letters.
  • Font border color: color for the letters border.
  • If we enable Show speech bubble:
    • Bubble background color: color for the dialog container.
    • Bubble border color: color tor the dialog container border.

Current voice

In this section, we can enable voice synthesizer. This option is still on development phase and only supports English voices.
