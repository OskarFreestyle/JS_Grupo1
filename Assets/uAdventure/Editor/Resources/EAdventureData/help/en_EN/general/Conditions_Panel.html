Conditions panel

In the conditions panel we can find a first part of it which details the conditions which are being edited. After that there is a bigger one where we can edit the conditions. In addition there is a button to add new conditions and accept the changes.

Top part: description of the condition

At this part of the panel we will find a brief description which corresponds to the condition we are editing.

Bottom part: add conditions and accept changes

At this part of the panel we will find two buttons. Through them we can add new conditions or just accept the changes introduced to the conditions we are editing, and closing the window.

The edition panel

In this panel we will find a very graphic description of the condition we are implementing. When we put the pointer over a condition we have the next following posibilities:

  • Remove: remove the condition.
  • Duplicate: creates a new condition just like the actual one, and afterwards the aforementioned condition joint by an "or".
  • Edite: edits the content of the condition. We can change the evaluated element or the value which is being compared as well. el valor con el que se compara.
  • When we put the pointer over a connection we can edit the value, which can oscillate between "AND" and "OR". The conditions will be groupped automatically depending on the meaning they have in the e-Adventure language.

Some other things...

When we add a condition, this will be added after the all existing conditions. If we want to add a condition in the middle position the best way to achieve this is chosing the previous condition to the place we want to insert the new one, duplicate it, and afterwards modificating the value of the new created condition.
