Advanced Features: Timers

A Timer allows a set of actions to be executed after a specified period of time.

In List of timers tab we can add new Timers by clicking on . Once one is selected we'll be able to edit the Time for this Timer and if we want it to appear during the game, clicking on Display in game option.

TIMER DOCUMENTATION

Information about the timer. It won't be used in the engine.

TIME

This section is only enabled if Display in game is enabled. Options are:

  • Display name: The name with which the timer shows in the screen.
  • Count-down: If enabled, time will be a count-down timer.
  • Show when stopped: If enabled, counter will show always, even if it is stopped.

LOOP CONTROL

In this section we edit the timer's behavior when it's stopped or ends a loop.

  • Multiple starts:
    • Set: Once the timer is stopped, it will start again once the conditions to start are met again.
    • Not set: Once the timer is stopped, it will not start again even if the conditions to start are met again.
  • Run in loops:
    • Set: The timer will run once and again until it is stopped.
    • Not set: The timer will run once and it will stop.

CONDITIONS TO START THE TIMER

In this section we can edit the conditions to start the timer. More info about conditions

CONDITIONS TO STOP THE TIMER

If we check Use end conditions timer will stop when these conditions are met. If not checked, timer will stop when conditions to start are not met.

EFFECTS

Finally, there are two section in which we can edit effects:

  • Effects to be executed if the timer expires
  • Effects to be executed if the timer is stopped
