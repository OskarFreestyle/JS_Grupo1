Effect move character

Through this effect we will be able to move the selected character to the selected position.

Create a move character effect:

  • Select the effect: Select the effect on the creation effects panel. Press OK.
  • Creation effect panel: afterwards an edition panel will appear:
    • In the first place we can move the aimed character. Afterwards we can select a scene from the chapter. As with other effects, the selection of the scene has the only purpose of preview, because the effect does not need to be related to the scene shown here.
    • The movement of the person selected will be executed in the scene where this effect was defined. Once you have chosen the scene, you choose the position that the character will move, either filling in the X and Y coordinates or by clicking on the chosen position on the stage
