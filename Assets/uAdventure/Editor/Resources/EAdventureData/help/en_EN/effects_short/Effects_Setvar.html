Effect set var

Through this effect we assign a value to the chosen value.

Create a set var effect:

  • Select the effect: Select the effect on the creation effects panel. Press OK.
  • Creation effect panel: afterwards an edition panel will appear:

    In this panel we can select the variable among the list of existing variables in the chapter. We can also create a new variable. For that purpose, just type the name of the new variable in the field text. Besides, after the text field of the variable selection, there is another field where we can edit the value which is going to be assigned to it.
