Effect deactivate flag

Through this effect we will deactivate the chosen flag.

Create a deactivate flag effect:

  • Select the effect: Select the effect on the creation effects panel. Press OK.
  • Creation effect panel: afterwards an edition panel will appear:

    In this panel we can select the effect that we want to deactivate among all the flags created in the chapter. We can create a new effect as well. Just by typing the name on the field text
