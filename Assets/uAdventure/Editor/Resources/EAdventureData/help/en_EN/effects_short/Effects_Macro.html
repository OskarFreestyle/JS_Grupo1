Effect launch macro

Through this effect a macro is executed, what means a simple set of default effects are launched. The effect launch all the effects assigned to the macro.

Create a launch macro effect:

  • Select the effect: Select the effect on the creation effects panel. Press OK.
  • Creation effect panel: afterwards an edition panel will appear: Select in this panel among the created macros for the proyect. We should highlight that we can not create this effect if we are selecting effects for a macro, or if there is no macro created.
