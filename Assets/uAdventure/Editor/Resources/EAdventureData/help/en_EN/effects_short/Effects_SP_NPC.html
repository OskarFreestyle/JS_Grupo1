Effect character says

With this effect a person will pronounce a phrase

Create an effect character says:

  • Select the effect: select the effect in the creation effects panel. Press OK.
  • Creation effect panel: aftwerwards an edition panel will appear:
    • In the first place is allowed to choose among the existent characters. When it has been choosen, the text field can be fill with the information required.
    • The text will appear with the colors associated to the conversations for the choosen character, and if the option "Always synthetize" is active, the frase will be synthetized.
