Player: Appearance

This tab only appears in third person adventures.

In this tab, we will define our main character representation. There are many animations to define (not all required).

In the top, we choose the Resources group, which is compound of a set of animations, we want to edit. In each group, we have a set of selectors for animations and preview panels for them.

If desired, we can create our own animation using the animation editor incorporated in the <e-Adventure> Editor clicking on Create/Edit in any of the resources.
