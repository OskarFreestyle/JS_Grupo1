This kind of Book can be added by clicking on . The following options are available:

  • Add title paragraph: a text element with a bigger font size will be added.
  • Add text paragraph: a text element with a normal font size will be added.
  • Add bullet paragraph: a bullet item will be add
  • Añadir image paragraph: an image will be added. The image can be chosen by clicking on Select... and deleted by clicking on .
