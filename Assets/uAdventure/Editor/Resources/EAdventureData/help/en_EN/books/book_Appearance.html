Book: Appearance

In Appearance tab the book aspect can be configured. We can customize the next elements:

  • Background image of the book: The Book content will be drawn above this image.
  • Image for left / right normal arrow: image for the left / right arrow (arrows to change the book page) when mouse is not over it.
  • Image for left / right over arrow: image for the the left / right arrow when mouse is over it..

NOTE: It is not necessary to define all images. For example, if we define only one side arrows, the opposites will be the symmetric ones.
