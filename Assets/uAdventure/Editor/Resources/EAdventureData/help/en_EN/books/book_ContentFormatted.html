Formatted book: Content

Content of this book type is built from HTML or RTF files. We can add new resources clicking on . In every resources, we have several options:

  • Type: asset type we want to add.
    • Local resource: HTML or RTF file which we add from a local file. We can also create a file using the <e-Adventure> Editor clicking on Create and then on Edit. This action leads to a simple HTML editor in which the content can be generated.
    • URL: select a remote file through its URL.
    • Image: resource is a local image file.
  • Asset: resource directory.
  • Margin: clicking on Edit Margins we open a dialog in which using sliders we will be able to place the content in relation to the background.
