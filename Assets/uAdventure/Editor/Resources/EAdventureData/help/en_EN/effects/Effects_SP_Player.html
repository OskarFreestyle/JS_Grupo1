Effect protagonist says

With this effect the protagonist says a phrase.

Creating an effect protagonist says:

  • Select the effect: select the effect in the creation effects panel. Press OK.
  • Creation effects panel: afterwards the next panel will appear:
  • In this panel we can fulfill the text field with the information is going to be said by the main character of the history. The text will appear with the associated colors to the conversation of the main characters, and if it have activated the option of always synthetize, the phrase will always be synthetized
