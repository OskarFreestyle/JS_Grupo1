Miscelanea effects

In this group we find some different effects

The effects we can find in this group are the following

  • Launch a macro: macros are a set of default effects created by the designer of the game. This effect launches all the effects defined on the macro.
  • Cancel action: this effect cancels the default effects of default actions. Use it to avoid an object being consumed automatically from the scene to execute a "Take" action, or to avoid an object being consumed from the inventory after performing a "give to" action.
  • Effect with probability: allows to choose between the execution of two effects depending on a probability, given a percentage. Through this effect we might create random behaviours on the adventures.
  • Wait time: The game action stops during the specified time. Used to emphasize certain aspects of the game, for example, used in combination with changes of scene, you can give the user a preview of a scene without being able to interact with the elements.
