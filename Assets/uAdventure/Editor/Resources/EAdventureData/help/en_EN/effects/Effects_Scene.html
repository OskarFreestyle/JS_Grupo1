Effect trigger scene

With this effect we switch to the selected scene.

Create a trigger scene effect:

  • Select the effect: Select the effect on the creation effects panel. Press OK.
  • Creation effect panel: afterwards the next panel will appear:

In the first place we can choose the scene we will switch once the effect is performed. Afterwards a preview of the scene is available and we can choose the position where the player will appear. This position can be chosen giving value to the coordinates X and Y, or just by clicking in the position of the desired scene.
