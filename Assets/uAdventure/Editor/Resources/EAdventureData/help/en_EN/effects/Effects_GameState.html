Game State

These effect allow to modify the game state by chaning flags and variables.

We can find the following effects in this group:

  • Activate: Activate the selected flag.
  • Deactivate: Deactivate the selected flag.
  • Set Value: Set the specified value on the selected variable.
  • Increment var: Increase in the selected units the chosen variable.
  • Decrement var: Decrement in the selected unit the chosen variable.
