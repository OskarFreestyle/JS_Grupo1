Plays animation effect

This effect reproduces an animation

Create a plays animation effect:

  • Select the effect: Select the effect on the creation effects panel. Press OK.
  • Creation effect panel: afterwards the next panel will appear:

In the first place we find a button to select the animation. Once you click "Select" a dialog to browes folders with a preview of selected animations appears. Once the file is selected, the button "View" will be activated, where you can also preview the animation. In this case is shown in a separate window.

If you want to delete the selected animation, you can choose another, or press the red X to the left.

The combo box below allows the selection of a scene. This scene is used in the preview panel in the center, where the place where the animation will appear can be selected. You can select the point either by clicking the preview scene or changing the coordinates.

Note that the choice of a scene while editing this effect is merely illustrative and does not mean that the effect will take place necessarily in that scene. The effect will be implemented when carrying out the action that triggers it, and will run in the scene where the character is at that time.
