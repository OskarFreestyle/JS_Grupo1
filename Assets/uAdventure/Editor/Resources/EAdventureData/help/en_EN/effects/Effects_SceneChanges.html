Effects to perform changes in the scene

These effects perform changes in the scene where player is located.

The effects in this group are the following:

  • Consum item: the selected object dissapeasr from the scene. This effect can be used associated to an action of "take", so to the player will look like the object has actually been taken.
  • Generate item: add the selected object into the inventory. Typical use is to add a recently grabbed object to the inventory.
  • Move player: moves the player to the selected position.
  • Move character: moves the selected character to the selected position.
  • Highlight item: highlights the selected item in the current scene.
