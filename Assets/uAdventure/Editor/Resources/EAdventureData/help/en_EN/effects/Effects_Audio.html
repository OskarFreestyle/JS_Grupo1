Effect play sound

This effect reproduces a sound

Create a play sound effect:

  • Select the effect: Select the effect on the creation effects panel. Press OK.
  • Creation effect panel: afterwards the next panel will appear:

In the first place we find the posibitily to play the sound in the background. If this option is not activated, no action will take place (launch more effects, moving the player, etc) till the sound is not completely reproduced.

Next to it we find a field to select the audio file. After pressing it, in addition to the posibility of browsing the directories we can play the file we select, so there is no need to leaving the application to hear it. Once the file is selected, the reproduction button will be active. Through this button we can also listen to the sound file.

If you want to delete the selection of the selected file, you can choose another, or press the red X to the left of the file name.
