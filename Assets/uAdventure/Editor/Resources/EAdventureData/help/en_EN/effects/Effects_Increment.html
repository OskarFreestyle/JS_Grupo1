Increase value effect

Through this effect a chosen variable is increased in the given units

Create an increase value effect:

  • Select the effect: Select the effect on the creation effects panel. Press OK.
  • Creation effect panel: afterwards the next panel will appear:

In the panel we can select the variable among the list of existant variables in the chapter or create a new one by typing in its name.

Next to variable selection panel, there is another field where we can edit the number of units we will increase the selected variable. We should highlight that those units will be add to the value which the variable has before the execution of the effect in the engine.
