Highlight Item effect

With this effect, an object will be highlighted or the highlighting on the object will be changed or removed.

Create a highlight item effect:

  • Select the effect: select the effect in the creation effects panel. Press OK.
  • Creation effect panel: aftwerwards the next panel will appear:

The first thing to select is the item to be highlighted. This effect will have no consequence if the object is not in the current scene. After the item is selected, the type of highlighting must be chosen: no highlight, blue highlight, red highlight, green highlight or black border highlight.

Finally, you have to select if you want the object to become animated or not.
