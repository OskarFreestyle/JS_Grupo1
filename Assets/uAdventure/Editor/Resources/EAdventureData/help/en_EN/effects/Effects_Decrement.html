Effect decrease variable

Through this effect the chosen variable is decreased in the indicated units.

Create a decrease variable effect:

  • Select the effect: Select the effect on the creation effects panel. Press OK.
  • Creation effect panel: afterwards the next panel will appear:

In this panel we can select the variable among an existent variable list from the chapter. We can create a new variable as well just by typing its name. Besides, inmediately after the variable section field, there is a different field where we can edit the number of units we will decrement the selected variable. Note that those units will be substracted to the value of the variable while the effect is being executed in the engine.
