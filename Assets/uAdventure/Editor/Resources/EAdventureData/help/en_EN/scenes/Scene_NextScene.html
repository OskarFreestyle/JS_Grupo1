Next Scene of the exit

Next Scene

In the combo panel the next scene can be used, the one which the exit is going to lead.

Use destination position

This option will only exit when the adventure is a third-person game. When the next scene is a regular scene, this field could be activated in order to choose where the character will appear. In the case this field is not activated, the character will appear in the default position.

Edit destination position

This option will only exit when the adventure is a third-person game. The button to edit the destination position allow the access to a panel where the player could be situated in the desired position.
