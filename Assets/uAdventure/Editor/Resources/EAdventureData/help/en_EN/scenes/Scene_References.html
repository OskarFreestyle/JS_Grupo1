Scene: reference to elements

This panel will set the references to elements (characters, objects or objects of atrezo) on the scene.

The order of the elements in the table determines the order in which they are painted at the scene. View information about the Layer

The column of cells is used to determine which reference is drawn and not in the panel below, does not affect the output of the game

The bottom panel

In the ottom panel the relative position of the objects in the scene can be modified, as well as it scale and, when corresponds, the influence area.

To see how to use the panel check the edition panel.
