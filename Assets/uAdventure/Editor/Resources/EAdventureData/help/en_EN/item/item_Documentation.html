Items: Documentation

In this tab, we can edit the item documentation. We can edit the next data:

  • Full description of the item: Info about the item. It won't be used in the engine.
  • Name of the item: Name for call the item.
  • Brief description of the item: This info will show when the item is watched.
  • Detailed description of the item: This info is shown whe the item is examined.

We also count with next check options:

  • Object returns to its original position after being dragged: if checked, item come back to its initial place after being dragged.
