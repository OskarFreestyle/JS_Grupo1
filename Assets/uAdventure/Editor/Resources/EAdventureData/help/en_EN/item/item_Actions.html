Items: Actions

actions table

We can define actions and their consequences (effects) for each item. To add a new action you click on . Availables actions are:

  • Normal actions: with Examine, Grab, Use.
  • Actions that need another element:
    • Use with: Item must be used with another one. Then, the effecs will be triggered. This item is choosen by a list.
    • Give to: Item must be used with a character. Then, the effecs will be triggered. This character is decided making a choice in a list which is displayed to the user.
  • Custom action: Custom action allows us defining one action which adapts to our requirements. Once selected, a dialog will ask us for a name for the action, (e.g.: buy, open, throw, etc. ) and then if the action needs of another item:
    • If we select Action, interaction with antoher item will not be necessary.
    • If we select Interaction, a dialog will ask us for the item which interacts with it.
    • Once decided, we can custom the button appearance for the action in the tab Personalization.

Each action is related, beyond some effects that will be explained later, some conditions that appear in the top table.

  • Needs get to: if it is necessary to reach and item to perform an action. We can set the minimum distance between the item and the main character necessary to perform the action. This condition only have sense if it is a third person adventure.
  • Conditions: in here we define the conditions which must be checked before the action is performed. If not, nothing will happen. More info about conditions.

Description and effects

Any action has a Full description for the action and Effects.

In the first, we can add all the information we want about the action.

In the second, clicking on we can add Effects for the action. More about effects.

Also, selecting Active when conditions are not and then clicking on Edit not-effects we can define the effects that will have when we perform the action but the conditions defined in top table are not met.
