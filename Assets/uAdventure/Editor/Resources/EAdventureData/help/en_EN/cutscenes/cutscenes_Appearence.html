Slidescene

Set of the slides of the slidescene

To choose Set of the slides of the slidescene we click on Select... and look for the file we want to our Cutscene. There are two file types we can use:

  • *.eaa: We can choose desired file clicking on Select... or create our own slide show.

    • <e-Adventure> Editor allows us creating or own Cutscenes using a simple Animation editor, clicking on Create/Edite, in which we can choose images forming the slide show, duration for every image, transitions between slides...

    See also: Help on Animation Editor

  • *_01.jpg: In this case, we have to name images forming our slide show in a special way .

    For example: if we have the next files: image_01.jpg, image_02.jpg, image_03.jpg, etc, <e-Adventure> Editor will take every image as slide for the slide show, showing them sorted by the number in name file.

We can preview slide show clicking on View and delete it clicking on .

Music of the scene

We can choose Music of the scene clicking on Select... . Admitted file types are: *.mp3, *.mid, *.midi. We can also listen to audio file into file navigator, before we select it.

Once selected, we can listen to it clicking on Play. To delete it, click on .
