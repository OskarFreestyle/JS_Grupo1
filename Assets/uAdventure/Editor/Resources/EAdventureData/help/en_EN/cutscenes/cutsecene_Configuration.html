Cutscenes: Cutscene end configuration

In Cutscene end configuration tab we can choose the behaviour of game action When cutscene end is reached:

  • Returns to the last scene: Game action go back to the scene where it was when the Cutscene was launched.
  • Goes to a new scene: Game action goes to a new scene. In ths mode, there are several options:
    • Id of next scene: We can choose next scene selecting it from a list.
    • Edit effects: We can add effects asociated with Cutscene end.
    • Use a destiny position for the player: We can choose the player position when he show up in next scene, enabling this option en clicking on Edit the destiny position.
    • Transition: kind of transitions between scenes.
    • Transition time: the time the transition will take.
  • Chapter ends: chapter finalizes.
